Python: Introduction

Kya hai Python? Kaisi hai? Kyun seekhein? — Sab kuch yahan

Python Kya Hai?

Python ek high-level, general-purpose programming language hai. Matlab — yeh language human ke liye easy hai (machine ke liye nahi), aur ise alag alag kamon ke liye use kar sakte hain.

Socho Python ek Swiss Army Knife ki tarah hai — ek hi cheez se website bhi bana sakte ho, AI model bhi train kar sakte ho, data analysis bhi kar sakte ho.

Python Syntax & Indentation

Python ke rules — in rules ko todoge toh error aayega!

Syntax Kya Hota Hai?

Syntax matlab bhasha ke rules. Jaise Hindi mein "Main school gaya" sahi hai, "School gaya main" thoda ajeeb lagta hai — lekin Python mein galat syntax matlab program nahi chalega.

Indentation — Python Ka Sabse Bada Rule

Python mein indentation (spaces/tab) mandatory hai! Yeh code ka structure batata hai. Doosri languages mein curly braces { } hote hain, Python mein nahi.

indentation_example.py
# SAHI — 4 spaces indentation
if 5 > 3:
    print("5 bada hai 3 se")   # 4 spaces aage
    print("Yeh bhi andar hai")  # same level
print("Yeh bahar hai")          # 0 spaces — if ke bahar

# GALAT — IndentationError aayega!
if 5 > 3:
print("Error!")  # Indent nahi kiya — ERROR!
❌ Common Mistake: Kabhi bhi tabs aur spaces mix mat karo! Ya to sirf 4 spaces use karo, ya sirf Tab. VS Code automatically handle karta hai.

Operators — Saare Prakar

Arithmetic, Comparison, Logical, Assignment, Membership, Identity — sab yahan

1. Arithmetic Operators — Ganit Ke Kaam

arithmetic.py
a, b = 10, 3

print(a + b)    # 13  — Jod (Addition)
print(a - b)    # 7   — Ghatao (Subtraction)
print(a * b)    # 30  — Gunao (Multiplication)
print(a / b)    # 3.333... — Float division
print(a // b)   # 3   — Floor division (poora number)
print(a % b)    # 1   — Remainder/Modulus (shesh)
print(a ** b)   # 1000 — Power (10 ki ghaat 3)

# Practical example — even/odd check
number = 7
if number % 2 == 0:
    print("Even")
else:
    print("Odd")   # Odd

2. Comparison Operators — Tulna Karna

comparison.py
x, y = 5, 10

print(x == y)    # False — barabar hai?
print(x != y)    # True  — barabar NAHI hai?
print(x > y)     # False — x bada hai?
print(x < y)     # True  — x chhota hai?
print(x >= y)    # False — x bada ya barabar?
print(x <= y)    # True  — x chhota ya barabar?

# Result hamesha True ya False aata hai

3. Logical Operators — and, or, not

logical.py
umar = 20
student = True

# and — dono True ho tab True
print(umar >= 18 and student)   # True and True = True
print(umar >= 18 and not student) # True and False = False

# or — koi ek True ho tab True
print(umar >= 18 or student)    # True or True = True
print(umar < 18 or student)     # False or True = True

# not — ulta kar do
print(not student)   # False
print(not False)      # True

4. Assignment Operators

assignment.py
x = 10       # Basic assignment
x += 5       # x = x + 5 = 15
x -= 3       # x = x - 3 = 12
x *= 2       # x = x * 2 = 24
x /= 4       # x = x / 4 = 6.0
x //= 2      # x = x // 2 = 3.0
x **= 3      # x = x ** 3 = 27.0
x %= 4       # x = x % 4 = 3.0
print(x)     # 3.0

5. Membership Operators — in, not in

membership.py
fruits = ["apple", "banana", "mango"]

print("mango" in fruits)       # True — list mein hai?
print("grape" in fruits)       # False
print("grape" not in fruits)   # True

# Strings ke saath bhi
text = "Python seekh raha hoon"
print("seekh" in text)     # True
print("Java" not in text)   # True

6. Identity Operators — is, is not

identity.py
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)     # True — values same hain
print(a is b)     # False — alag objects hain memory mein
print(a is c)     # True — same object!
print(a is not b) # True

# None check ke liye 'is' use karo:
val = None
print(val is None)   # True — recommended way
🔑 Key Difference: == values compare karta hai (same content?), jabki is memory address compare karta hai (same object?).

Expressions — Values Ka Combination

Expression matlab ek ya zyada values ko operators se jod ke ek naya value banana. Jab Python kisi expression ko evaluate karta hai, toh ek single value milti hai.

expressions.py
# Arithmetic Expression — result ek number hota hai
result = 2 + 3 * 4    # 14 — BODMAS rule lagta hai! (3*4 pehle)
result2 = (2 + 3) * 4  # 20 — brackets se override karo

# Boolean/Relational Expression — result True ya False hota hai
check = 10 > 5         # True
check2 = (5 + 3) == 8  # True

# String Expression — text jodna
naam = "Rani"
greeting = "Hello, " + naam + "!"   # Hello, Rani!

# Complex expression — sab ek saath
umar = int(input("Umar batao: "))
adult = umar >= 18 and umar < 60   # True agar 18-59 ke beech hai
print(f"Working age adult: {adult}")

# Operator precedence (BODMAS jaisi priority)
# ** → *, /, //, % → +, - → Comparison → not → and → or
print(2 + 3 ** 2)        # 11 — pehle 3**2=9, phir 2+9
print(10 - 2 * 3 + 1)   # 5 — pehle 2*3=6, phir 10-6+1
📌 Expression vs Statement: Expression kuch return karta hai (jaise 2 + 3 returns 5). Statement ek action karta hai (jaise print("hello")). Dono milkar program banata hai!

range_examples.py
# range(stop) — 0 se start, stop se pehle
print(list(range(5)))         # [0, 1, 2, 3, 4]

# range(start, stop)
print(list(range(1, 6)))      # [1, 2, 3, 4, 5]

# range(start, stop, step)

while Loop — Condition Tak Chalta Rahe

while_loop.py
# Basic while
count = 1
while count <= 5:
    print(f"Count: {count}")
    count += 1    # Increment nahi kiya toh infinite loop!

# Password check game
password = "python123"
attempts = 0

while True:
    guess = input("Password enter karo: ")
    attempts += 1
    if guess == password:
        print(f"Sahi! {attempts} try mein nikala")
        break
    elif attempts >= 3:
        print("Bahut galat — locked!")
        break
    else:
        print("Galat! Phir koshish karo")
⚠️ Infinite Loop Warning: while loop mein koi condition update karna mat bhulo! Agar condition kabhi False nahi hogi toh program hamesha chalta rahega. Rokne ke liye Ctrl+C dabao.
Unit 2 · Topic 9

Loop Control — break, continue, pass

Loop ko rokna, skip karna, ya kuch nahi karna

break — Loop Band Karo

break_example.py
# Pehla negative number mila toh ruk jao
numbers = [5, 8, 3, -2, 9, 1]

for num in numbers:
    if num < 0:
        print(f"Negative mila: {num} — ruk gaye!")
        break           # Loop yahan band
    print(f"Number: {num}")

# Output:
# Number: 5
# Number: 8
# Number: 3
# Negative mila: -2 — ruk gaye!

continue — Yeh Wala Skip Karo

continue_example.py
# Sirf even numbers print karo
for i in range(1, 11):
    if i % 2 != 0:
        continue    # Odd hain? Skip! Agle pe jao
    print(i, end=" ")
# Output: 2 4 6 8 10

# Empty strings skip karo
names = ["Riya", "", "Amit", "", "Priya"]
for name in names:
    if name == "":
        continue
    print(f"Hello, {name}!")

Recursion — Function Jo Apne Aap Ko Call Kare

⚠️ Warning: Recursion mein Base Case (kab rukna hai) hona bahut zaroori hai, warna infinite loop ho jayega!
recursion_example.py
# Factorial — n! = n * (n-1) * (n-2)... * 1
def factorial(n):
    if n == 1:           # Base Case (Rukne ki condition)
        return 1
    else:
        return n * factorial(n - 1)   # Recursive Call

print(factorial(5))    # 5 * 4 * 3 * 2 * 1 = 120

# Sum of n numbers
def sum_n(n):
    if n <= 0:
        return 0
    return n + sum_n(n - 1)

print(sum_n(10))       # 55

File Handling mein Try-Except

safe_file.py
def safe_read_file(filename):
    """Safe tarike se file padho"""
    try:
        with open(filename, "r", encoding="utf-8") as f:
            content = f.read()
            print(f"File mili! Size: {len(content)} characters")
            return content
    except FileNotFoundError:
        print(f"Error: '{filename}' nahi mili")
        return None
    except PermissionError:
        print(f"Error: '{filename}' padhne ki permission nahi")
        return None
    except Exception as e:
        print(f"Koi aur error: {e}")
        return None

# Use karo
